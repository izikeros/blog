List Unlinked (Orphaned) Notes in Obsidian

Table of Contents

    Here is how you can create dynamic list of unlinked notes (files) using dataview

    

    LIST FROM ""
WHERE length(file.inlinks) = 0 AND length(file.outlinks) = 0
