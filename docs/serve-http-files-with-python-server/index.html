Serve HTTP files with python server

Sometimes there is a need to serve HTML page instead of just displaying it in the browser: - share content with others (allowing download) - serve HTTP document with javascript when javascript can't be read due to CORS limitations.

From the directory with the content you want to serve type in terminal:

python -m http.server 9000
