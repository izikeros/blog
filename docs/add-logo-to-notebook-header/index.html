Add logo to notebook header cell

At the top of the first markdown cell (e.g. with notebook description) add HTML code:

<div class="row">
  <div class="col-md-8" markdown="1">
      <h1>Notebook title</h1>
  </div>
  <div class="col-md-4" markdown="1">
      <img height="600px" class="center-block" src="../../gfx/project_logo_150px.jpg">
  </div>
</div>

Source: Have two columns in Markdown

NOTE: - replace Notebook title and ../../gfx/project_logo_150px.jpg with your data. - you can change the relative column widths by changing the 8 and/or 4 shown in the code above, just make sure the sum of those two numbers is always 12.
