Top-ranked ZSH plugins

Posted on Thu 22 March 2018 in Posts • 2 min read

ZSH plugins that have highest count of Github stars

There is exhaustive but curated list of ZSH plugins posted on Github project Awesome ZSH plugins. You can find there 800+ links to plugins, themes and zsh plugin managers/frameworks. Even though it is collection of awesome stuff the number is bit high get orientation which plugins gained already good reputation from ZSH users comunity. This post aims at identyfying most popular plugins where pupluarity is measured wuth number of stars that github users added to given plugin. For clarity I will focus here only on plugins excluding frameworks and themes that are also listed on "Awesome ZSH plugins" website.

What are Github stars?

Stars is the way how users can 'bookmark' projects. The stargazers statistics are available via GithubAPI
